# 🚀 クイックスタートガイド

## 📦 インストール

### 1. 必要な環境
- Python 3.8以上
- pip (Pythonパッケージマネージャー)

### 2. セットアップ手順

```bash
# 1. プロジェクトディレクトリに移動
cd textbook_splitter

# 2. 依存パッケージをインストール
pip install -r requirements.txt

# 3. アプリを起動
streamlit run app.py
```

ブラウザが自動的に開き、アプリが表示されます。

## 📖 使い方

### ステップ1: ファイルをアップロード
- 「Browse files」ボタンをクリック
- `.md` または `.txt` ファイルを選択

### ステップ2: オプション設定
- 必要に応じて「ページ表記 [ページ x] を削除する」にチェック

### ステップ3: 分割処理を実行
- 「🔄 分割処理を実行」ボタンをクリック

### ステップ4: 結果を確認
- 生成されるファイル一覧が表示されます
- 各ファイルの行数と文字数も確認できます

### ステップ5: ダウンロード
- 「📥 ZIPファイルをダウンロード」ボタンをクリック
- `textbook_sections.zip` がダウンロードされます

## 📁 出力ファイルの例

```
textbook_sections.zip
├── 00_導入.txt
├── 01_産業革命と工業化.txt
├── 02_アメリカ合衆国とラテンアメリカ諸国の独立.txt
```

## 🔧 トラブルシューティング

### アプリが起動しない場合
```bash
# Streamlitのバージョンを確認
streamlit --version

# 再インストール
pip install --upgrade streamlit
```

### ファイルが読み込めない場合
- ファイルがUTF-8エンコーディングであることを確認
- `.md` または `.txt` 拡張子であることを確認

### 分割結果が期待と異なる場合
- テキスト内に `###` で始まる見出しがあることを確認
- 見出しの前に空白がないことを確認

## 💡 ヒント

- **大きなファイル**: 200MBまでのファイルをアップロード可能
- **複数ファイル**: 一度に1つのファイルのみ処理可能
- **プレビュー**: ダウンロード前にファイル一覧で内容を確認できます

## 📞 サポート

問題が発生した場合は、以下を確認してください:
1. README.mdの詳細な説明
2. test_split.pyでテスト実行
3. エラーメッセージの内容
